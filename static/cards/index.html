Authenticator creation

Please register your card number and expiration date, we will then create and associate your authenticator. No payment will be proceed.

* We only accept MasterCard, Visa, American Express, JCB, Diners Club and Discover. * This is a demonstrator no payment will be processed.

How to register

By clicking OK, we will create your authenticator. You will have to perform your U2F gesture to do so.

If you are using chrome as a navigator, please go to chrome://flags and enable the Web Authentication API flag.

If you already have an U2F dongle, prepare it. For exemple, plug your FIDO USB device.

Tap it after clicking OK

If you don't have any U2F dongle, you can use SoftU2F as a software authenticator (MacOS only). You can lean how to use it by a quick Readme overview.
